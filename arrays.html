Arrays in JavaScript

Introduction to Arrays

Arrays are simply a list of items. For example:

List of Things I need to do (Todo list
1. Eat
2. Code
3. Sleep
4. Repeat

Can be represented as an array in JavaScript in the following manner:

var todoList = ['Eat', 'Code', 'Sleep', 'Repeat',];

When using arrays, there are certain things we can do with the items in an array which includes display, add, change and delete items.

Code Examples

1. Creating and displaying an array
Code: 
// Create and store array in a variable
var todos = ['Todo 1', ' Todo 2', ' Todo 3'];
// Display array
document.write('My todos are: ', todos);
2. Adding an item or group of items to an array
Code: 
// Create and store array in a variable
var todos = ['Todo 1', ' Todo 2', ' Todo 3'];
// Adding items to the array
todos.push(' Todo 4', ' Todo 5');
// Display new array
document.write('My todos are: ', todos);
3. Changing or updating an item in an array
Code: 
// Create and store array in a variable
var todos = ['Todo 1', ' Todo 2', ' Todo 3'];
// Change item in array
todos[0] = 'First todo updated';
// Display new array
document.write('My todos are: ', todos);
4. Deleting an item from an array
Code: 
// Create and store array in a variable
var todos = ['Todo 1', ' Todo 2', ' Todo 3'];
// Delete an item in an array
todos.splice(1, 1);
// Display new array
document.write('My todos are: ', todos);
