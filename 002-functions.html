Functions in JavaScript

Introduction to functions

Functions are just recipes. For example:

How to make Turkey Sandwich.
1. Get one slice of bread,
2. Add turkey, and
3. Put a slice of bread on top.

Can be represented as a function in the following manner:

function makeTurkeySandwich() {
    Get one slice of bread;
	Add turkey;
	Put a slice of bread on top; 
}
makeTurkeySandwich();

Customizing Functions

Functions can be customized. For example:

How to make Sandwich ____
1. Get one slice of bread,
2. Add ____, and
3. Put a slice of bread on top.
function makeSandwichWith(filling) {
	Get one slice of bread;
	Add filling;
	Put a slice of bread on top;
}
makeSandwichWith("ham"); // Where the filling is ham

Code Examples

1. A function that says hi to a person
Code: 
function sayHiTo (personName){
	document.write("Hi ", personName);
}
sayHiTo("SimDave");
2. A function that displays an array
Code: 
var myArray = ['item 1', ' item 2', ' item 3'];
function displayArray(){
	document.write('Array: ', myArray);
}
displayArray();
3. A function that adds an item to an array
Code: 
var myArray = ['item 1', ' item 2', ' item 3'];
function addItem(){
	myArray.push(" item 4", " item 5");
	document.write('Array: ', myArray);
}
addItem();
OR
Code: 
var myArray = ['item 1', ' item 2', ' item 3'];
function addItem(item){
	myArray.push(item);
	document.write('Array: ', myArray);
}
addItem(" item 4, item 5");
3. A function that changes an item in an array
Code: 
var myArray = ['item 1', ' item 2', ' item 3'];
function changeItem(position, newValue){
	myArray[position] = newValue;
	document.write(myArray);
}
changeItem(1, " changed item");
4. A function that deletes an item in an array
Code: 
var myArray = ['item 1', ' item 2', ' item 3'];
function deleteItem(position){
	myArray.splice(position, 1)
	document.write(myArray);
}
deleteItem(1);
