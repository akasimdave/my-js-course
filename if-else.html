If Else Statements in JavaScript

Introduction to if-else statements

In JavaScripts, as in other programming languages, you'll deal with conditional scenarios where you have to make decisions based on certain criteria. This can be achieved using the if-else conditional statements.

Code Examples

1. A program that gives access to people between ages 18 and 24
Code: 
function programAccess(age){
if (age >= 18 && age <= 35) {
	document.write("Access granted");
}
else if (age < 18) {
	document.write("You're too young for the program.");
}else{
	document.write("You're too old for the program.");
}
}
programAccess(37);
